# 원페이지 랜딩 스타터 (완전 무료 배포용)

## 구성
- `index.html` : 랜딩 본문
- `styles.css` : 최소 스타일
- `script.js` : 편의 기능

## 내 정보로 바꾸는 곳
- 전화번호, 카카오/이메일 링크
- 주소/지도 임베드
- 구글폼 임베드(또는 타입폼 등)
- `<title>`, `<meta name="description">`
- (선택) 네이버 서치어드바이저/구글 애널리틱스 스니펫

## 10분 배포 (GitHub Pages)
1) 깃허브 가입 → 새 저장소(repo) 생성 (공개)
2) 이 폴더 3개 파일을 그대로 업로드
3) 저장소 Settings → Pages → Source: `Deploy from a branch` → `main /(root)` 선택
4) 1~2분 후 `https://사용자명.github.io/저장소명` 주소로 접속

## 커스텀 도메인 연결(선택)
- 도메인 구매 후(DNS 관리에서) A/CNAME을 GitHub Pages 안내값으로 설정
- SSL은 자동(최대 수분 소요)

## 네이버/구글 검색 등록(선택)
- 네이버 서치어드바이저, 구글 서치콘솔에서 소유 확인 메타 태그 추가
- 사이트맵은 원페이지 기준 필수는 아님

## 광고 랜딩으로 쓸 때 팁
- 핵심 정보 상단(전화/문의/혜택/가격 요약)
- 이미지/동영상은 용량을 줄여서 업로드 (LCP 지표 개선)
- 폰트는 웹 안전 폰트 또는 1~2종만
- 정책/법적 고지(사업자번호, 개인정보처리방침) 하단 표시
